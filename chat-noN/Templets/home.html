University Chatbot Project

Developed using NLTK, December 2024

About this Project We have developed a chatbot that serves as a virtual assistant for students at the university. The chatbot is designed to help students with queries about courses, exam schedules, campus facilities, and more. This project was built using the Natural Language Toolkit (NLTK) in Python.

The chatbot is intended to be an interactive and user-friendly tool, making it easier for students to access information and get their questions answered quickly.
